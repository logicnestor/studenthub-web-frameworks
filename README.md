# StudentHub

StudentHub is a web-based Student Management System designed as part of the Web Development Frameworks course at CHARUSAT.

---

## Project Objectives

- Student Management
- Faculty Management
- Attendance Tracking
- Assignment Submission
- Study Materials
- Role-Based Dashboards
- Responsive Design

---

## Project Modules

### Student

- Dashboard
- Attendance
- Assignments
- Study Materials
- Profile

### Faculty

- Dashboard
- Attendance
- Upload Materials
- Assignments

### Admin

- Dashboard
- Student Management
- Faculty Management
- Department Management
- Reports
